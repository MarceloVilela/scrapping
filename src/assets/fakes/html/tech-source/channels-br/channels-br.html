README.md

Canais Brasileiros para Pessoas Desenvolvedoras

Repositório responsável em listar Canais no Youtube ou Streaming sobre Tecnologia, Desenvolvimento e Programação em Português.

Esse projeto foi baseado no projeto Awesome

Agradecimentos especiais para Glaucia e Lucas que se propuseram em ajudar com ótimas ideias e modificações ❤️

Contribuições 📌

Pedimos, por favor, que dêem uma olhada nas Diretrizes de Contribuição antes. E desde já, agradecemos a todos os contribuidores. Vocês são demais! ❤️❤️

Para propor melhorias nessa lista, basta abrir uma issue nesse repositório. Assim, todos poderão colaborar para o melhor desenvolvimento desse repositório. E sintam-se à vontade em fazer Pull Requests!!
